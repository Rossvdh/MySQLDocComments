GetSeriesID

Returns an int with the ID number of the series that the given book is in

Parameters:

bookID INT ID number of the book

Returns: INT, ID number of the book's series

IsFiction

Returns 1 if the given Book is in the Fiction table, 0 otherwise

Parameters:

idNum Int ID number of the Book

Returns: int, 1 if the book is Fiction, 0 otherwise.

addAuthor

Adds an Author with the given name to the Authors table. Automatically inserts an ID number

Parameters:

name VARCHAR(100) the author's full name

searchAuthorID

Returns the books with which the given Author has a role

Parameters:

author Int the Author's ID number

Column set:

id title id name role

searchDateGreater

Returns the books bought after the given date. After such that mnth > m, and yr >= y

Parameters:

m INT minimum month
y INT minimum yr

Column set:

title author price mnth yr
